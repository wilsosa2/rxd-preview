# RxD Netlify Preview

This is a Vite + React + Tailwind starter preview for the RxD current-site clone.

## Local run

```bash
npm install
npm run dev
```

## Deploy to Netlify

Use these settings:

- Build command: `npm run build`
- Publish directory: `dist`

You can drag-and-drop the whole project into a Git repo and connect it to Netlify, or upload the built `dist` folder after running `npm run build`.
